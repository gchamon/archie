These docs are tables that document in a readable and centralized manner the
keyboard shortcuts available in my hyprland setup.

In order to update the docs, you need to take into consideration only the
shortcuts configured in ./hypr/hyprland.conf.

If you don't know the syntax for hyprland you should panic with a suitable message.

The documentation is located at ./docs/KEYBOARD_SHORTCUTS.md.

When updating the documentation make sure to follow the structure already
present, keeping each entry consistent with the rest of the document.
